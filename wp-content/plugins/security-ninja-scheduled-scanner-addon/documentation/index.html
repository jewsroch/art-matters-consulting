B) Installation - top

Installation

  1. Download the ZIP package from CodeCanyon.
  2. Open WordPress admin and go to Plugins -> Add New -> Upload. Browse for the ZIP file security-ninja-scheduled-scanner-plugin.zip on your computer and hit "Install Now".
  3. Activate the plugin.

Post install 5sec tutorial

  1. Open Tools - Security Ninja
  2. Click the "Scheduled Scanner" tab, set scan settings and click "Save Changes"

Upgrade

  1. Deactivate and delete the plugin in your WP admin -> Plugins.
  2. Use the Add New -> Upload function to upload the new plugin ZIP file.
  3. Activate the plugin.

Core Scanner requires Security Ninja plugin v1.45, optionally Security Ninja Core Scanner 1.20 and WordPress core v3.4.

B) Usage - top

WordPress cron function depends on site visitors to regularly run its tasks. If your site has very few visitors the tasks wont be run on a regular, predefined interval. Wptuts+ has a great article explaining how to make sure the cron does get run even if you have very few visitors.

Options

  • scan settings - enables/disabled scheduled scans and configures which modules it runs
  • scan schedule - sets the scan frequency; twice daily is enough but you can set it to hourly as well
  • email report - email reports are optional and can be set after every scan or just after does that indicate a change on the site
  • email recipient - email address of the person who will get the reports

After saving options it's recommended to test them using the "Test settings" button. Scans will be immediately run, added to the log and emails sent if they are configured so.

Scan Logs

Logs show the last 50 scan runs (if you need more edit the WF_SN_SS_LOG_LIMIT constant). Each records shows the time-stamp, how long it took for the scan to run and details for both Security Ninja and Core Scanner add-on if they're both enabled.
If needed you can delete all logs with the "Delete all log entries" button.

D) FAQ - top

Will this plugin slow my site down?

Absolutely not. The cron runs in the background and it is not run that frequently to cause any issues.

Will it work on my theme?

Sure! Security Ninja Scheduled Scanner works with all themes.

What changes will Security Ninja Scheduled Scanner make to my site?

None! It only scans and shows you reports. It does not do any changes.

Is this plugin safe to use?

Of course.

Is this plugin legal to use?

Yes. It's your site you can do whatever you want with it. Running tests on other people's sites is illegal but Security Ninja Core Scanner can only perform tests on the WP it's installed on.

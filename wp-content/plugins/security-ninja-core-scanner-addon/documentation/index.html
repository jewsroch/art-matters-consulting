B) Installation - top

Installation

  1. Download the ZIP package from CodeCanyon.
  2. Open WordPress admin and go to Plugins -> Add New -> Upload. Browse for the ZIP file sn-core-scanner-plugin.zip on your computer and hit "Install Now".
  3. Activate the plugin.

Post install 5sec tutorial

  1. Open Tools - Security Ninja
  2. Click the "Core Scanner" tab and then "Scan core files" to analyze your site's core files

Upgrade

  1. Deactivate and delete the plugin in your WP admin -> Plugins.
  2. Use the Add New -> Upload function to upload the new plugin ZIP file.
  3. Activate the plugin.

Core Scanner requires Security Ninja plugin v1.2 and WordPress core v3.3.

B) Usage - top

After files are scanned they will be sorted into 5 lists. Each list shows the files in it as well as an explanation and actions that you can make. Please be careful when restoring files as there is no undo!

Files are sorted into following lists:

  • files that have been modified and should not have been
  • files that are missing and should not be
  • files that have been modified and they are supposed to be
  • files that are missing but they are not vital
  • files that are intact

D) FAQ - top

Will this plugin slow my site down?

Absolutely not. You may experience a slight slow down while the files are scanned but that takes less than a minute.

Will it work on my theme?

Sure! Security Ninja Core Scanner works with all themes.

What changes will Security Ninja Core Scanner make to my site?

On its own - none! Only when you choose to restore a modified will it will do so.

Is this plugin safe to use?

Of course.

Is this plugin legal to use?

Yes. It's your site you can do whatever you want with it. Running tests on other people's sites is illegal but Security Ninja Core Scanner can only perform tests on the WP it's installed on.
